# Validation Ground Truth Data

This directory contains real-world benchmark data collected from published sources
for validating the UrbanEV Maryland simulation results.

## Sources
- INL EV Project (2015) — 8,300 vehicles, 6M charging events
- NREL Public Charging Utilization (2024)
- DOE AFDC Infrastructure Trends (2024)
- Maryland PSC EV Pilot Data
- US DOE/EIA Electricity Rates
- EPA Vehicle Energy Consumption
- FHWA VMT Statistics
